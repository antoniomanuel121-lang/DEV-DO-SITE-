  - 60×60 — 19 coleções

  - 80×120 — 5 coleções

  - 120×180 — 3 coleções
  - Faucet — 1 coleção

  - Sanitary — 1 coleção

- Preparado para receber imagens reais extraídas dos PDFs e links Dropbox em versões futuras.

PDFs ficam sempre na Dropbox. O GitHub guarda apenas código e imagens.
